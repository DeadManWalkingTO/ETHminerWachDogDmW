## Usage
* Download or Clone
* Unzip
* Copy ETHminerWachDogDmW.bat (Windows) or ETHminerWachDogDmW.sh (Linux) to ETHminer folder.
* Modify the "Execution Code" section with your commands.
* Run the script.
